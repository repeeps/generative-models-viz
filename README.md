# 생성모델, 눈으로 보기

기계가 데이터를 <b>알아보는</b> 걸 넘어 한 번도 없던 것을 <b>새로 그려내는</b> 법을, 수식·코드 없이 **움직이는 그림과 인터랙션**으로 익히는 비영리 학습 자료입니다. 0번 입문 + 16챕터. (지도학습 기초·트랜스포머는 별도 트랙)

> 핵심 한 문장 — **데이터의 분포를 배워, 거기서 새 표본을 뽑는다.** VAE·GAN·Diffusion은 이 한 문장을 다르게 푼 세 갈래예요. 진짜 사진 대신, 좌표가 매끄럽게 모양으로 바뀌는 **글리프**(작은 얼굴)를 생성물의 대역으로 써요 — 원리는 같고 눈으로는 더 또렷해요.

## 어떻게 보나

별도 빌드·설치가 없습니다. `index.html`을 브라우저로 열면 됩니다. 모든 분포·샘플링·잠재공간 보간·노이즈/디노이징·GAN 경쟁·가이던스·FID 비교는 브라우저에서 실시간 계산·렌더링됩니다.

## 구조

처음이라면 **0번부터 순서대로**. 척추는 **생성이란(A) → VAE(B) → GAN(C) → Diffusion(D) → 평가·종합(E)**. 두 기둥은 **VAE(B)** 와 **Diffusion(D)**, GAN(C)은 그 사이 다리예요.

- **0번 — 만들어내는 기계**: 알아보기 vs 만들기 · 분포 배워 뽑기 · 글리프 약속 · 코스맵
- **PART A · 생성이란 무엇인가**
  - 01 판별 vs 생성 (경계 긋기 vs 분포 그려 뽑기, p(x))
  - 02 분포에서 표본 뽑기 (샘플링 · '새로움' vs 베끼기)
  - 03 잠재공간 (의미 손잡이 · 보간 — '꿈'으로 제시, 05에서 갚음)
- **PART B · 잠재변수 모델 — VAE** (기둥①)
  - 04 오토인코더 (압축→병목→복원, 첫 잠재공간)
  - 05 AE의 한계 → VAE (빈자리의 쓰레기 → 분포로 채워 매끄럽게)
  - 06 주사위를 밖으로 (reparam z=μ+σ·ε · 보간 생성 · 흐릿함=평균)
- **PART C · 적대적 생성 — GAN** (다리)
  - 07 위조범 vs 감별사 (생성자·판별자 경쟁 · 균형 50%)
  - 08 학습의 줄다리기 (불안정 · 모드 붕괴 = 과적합의 사촌)
- **PART D · 확산모델 — Diffusion** (기둥②)
  - 09 노이즈로 무너뜨리기 (forward — 고정 레시피, 학습 0)
  - 10 노이즈 걷어내기 (reverse 한 스텝 · 노이즈=클린 등가 · 배운 근사)
  - 11 노이즈를 맞혀라 (학습=회귀, 정답은 직접 섞은 노이즈 · 한 모델+t)
  - 12 한 장을 그려내기 (노이즈→반복 디노이징 · 출발이 달라 다양)
  - 13 조건 주기 (텍스트→이미지 · 가이던스 손잡이: 충실도↔다양성↔깨짐)
- **PART E · 평가 · 종합 · 너머**
  - 14 잘 만들었나? (품질×다양성 · 베끼기 · FID=두 무리 분포 비교)
  - 15 한 흐름 종합 (VAE/GAN/Diffusion 비교표 · 무엇을 쓸까)
  - 16 너머 · 회고 (잠재확산[B+D 합체] · 자기회귀 생성 · 회고)

## 기술

- 순수 정적 HTML + JavaScript (의존성·빌드 없음)
- 공통 엔진 `shared/lib.js`(VZ 코어 + 스텝 플레이어 VZ.AL) + `shared/lib-gen.js`(생성모델 도메인 엔진 VZ.GEN: 좌표→매끄럽게 모핑되는 글리프 디코더·노이즈 낀 글리프·잠재공간 격자·분포 곡선·점구름·밀도·인코더/디코더 모래시계·게이지·비교표) — 모두 순수 함수(상태 → SVG)
- 다크 테마 `shared/theme.css`
